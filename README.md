# EITANGOroppy

個別指導塾のアルバイト中に、生徒の「単語暗記がつまらない・続かない」という課題を解決するために開発した英単語学習Webアプリです。
生徒やスタッフへのヒアリングを重ね、「生徒が続けたくなる仕組み」にこだわって生成AIを活用しながら開発・改善を繰り返し、実際に塾へ導入しました。
※収録されている英単語データは、動作テスト用に生成したサンプルです。
[roppy](https://mtks062317-stack.github.io/EITANGOroppy/)
✨ 主な機能
4つの学習モード

スライドショーモード：単語が自動で切り替わり、音声読み上げ付きで流し見学習ができる
単語帳モード：自分のペースでめくれるカード形式
テストモード：4択クイズ形式。範囲ごとに前回スコア・最高スコアを記録
リターン学習：間違えた問題だけを自動で集めて復習。正解するまで繰り返し出題

続けたくなる仕組み

🔥 コンボシステム：連続正解でコンボ演出（5/10/15/20コンボで特別演出）
🎉 エフェクト：正解時のパーティクル、合格時の紙吹雪
🎵 サウンド：Web Audio APIで自作した効果音・BGM
💬 応援メッセージ：起動するたびに変わる格言・応援ボード

カスタマイズ・設定

🌙 ダークモード対応
🔊 単語の音声読み上げ（発音確認）
🎯 合格レベル設定（60%〜100%、講師用パスワード付き）
💾 学習データの自動保存（範囲ごとのスコア・合格状況）

🛠 使用技術

HTML / CSS / JavaScript（1ファイル完結型）
Web Audio API（効果音・BGMの自作）
Web Speech API（単語読み上げ）
Canvas API（パーティクルエフェクト）
localStorage（学習データの永続化）
GitHub Pages（ホスティング）

📝 開発の背景
塾講師として働く中で、紙の単語帳では生徒の集中が続かず、学習効率が上がらないという課題を感じていました。「もっと楽しく効率的に暗記したい」という生徒の声をもとに、プログラミング知識ゼロの状態から生成AIを活用して開発をスタート。
リリース後も生徒・スタッフへのヒアリングを続け、「ここが使いにくい」「この機能が欲しい」という声をもとにUI改善や機能追加を300回以上重ねました。間違えた問題だけを復習できるリターン学習や、ゲームのようなコンボ演出は、すべて現場の声から生まれた機能です。
